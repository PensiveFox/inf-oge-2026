# 🌳 Метод динамического программирования (Дерево вниз головой)

**Визуализация метода III для решения задания 9 ОГЭ 2026**

## 📋 Описание метода

**Метод динамического программирования** - это классический способ решения задачи подсчёта путей в DAG (направленном ациклическом графе).

### Принцип "Дерево вниз головой"

```
Начало (A)                    [1 путь]
   ↓     ↓     ↓
   B     C     D              [считаем пути через B, C, D]
   ↓     ↓     ↓
   E     F     F              [суммируем пути]
   ↓     ↓
   G     G
   ↓
Конец (H)                     [итоговое количество путей]
```

### Алгоритм

1. **Начало:** Вершина A имеет 1 путь (сам путь из A)
2. **Для каждой следующей вершины:**
   - Количество путей = СУММА путей из всех предыдущих вершин, из которых есть ребро
3. **Конец:** В вершине H получаем итоговое количество путей

## 🎯 Пример

```
Граф:
A → B → E → H
A → C → E → H  
A → D → H

Подсчёт:
A: 1 путь (начало)
B: 1 путь (из A)
C: 1 путь (из A)
D: 1 путь (из A)
E: 2 пути (из B=1 + из C=1)
H: 4 пути (из E=2 + из D=1 + еще...)
```
